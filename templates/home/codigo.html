Código

Uno de los objetivos de este proyecto es fomentar el uso de la aplicación que se utilizó para hacer los Mapas de Medios de Chile y Colombia en la creación de otros mapas en otros países de la región y del mundo. Para esto hemos creado un repositorio público en Github, desde donde se puede descargar el código necesario para hacer una aplicación/sitio web de este tipo, y el código de Mapa de Medios en particular.

Los repositorios son:

  • Mapa de Medios: Repositorio donde encontrará el código completo de este proyecto y la explicación sobre cómo usarlo. Si necesita orientación, podemos ayudarle.
  • Media Mapper: Repositorio con las instrucciones detalladas sobre los requisitos y la forma de proceder para crear y mantener un proyecto de este tipo.

Esta aplicación fue desarrollada en Django (Python Framework), con una base de datos MySQL y corre desde un servidor Apache.
